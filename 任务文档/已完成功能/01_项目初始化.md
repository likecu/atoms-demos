# 项目初始化 (已完成)

## 技术栈配置
- **框架**: Next.js 14 App Router (React 19)
- **语言**: TypeScript (严格模式)
- **样式**: Tailwind CSS v4 + Shadcn UI
- **图标**: Lucide React

## 基础设置
- `package.json`: 定义了核心脚本 `dev`, `build`, `start`, `lint` 以及 Docker 相关命令。
- `tsconfig.json`: TypeScript 配置文件，确保类型安全。
- `tailwind.config.ts`: 样式配置，集成 Shadcn 插件。
- `components.json`: Shadcn UI 组件库配置。
- `.env.local`: 环境变量配置 (Supabase, OpenRouter Key)。
