# 布局与UI实现 (已完成)

## 核心布局组件
使用 `react-resizable-panels` 实现灵活的三栏布局，支持拖拽调整宽度。

### 1. 侧边栏 (Sidebar)
*   **左侧面板**：包含项目历史记录、新建聊天入口、用户菜单。
*   **功能**：虽然历史记录数据部分为 Mock，但结构已完成，支持折叠。
*   **文件**: `components/layout/sidebar.tsx`

### 2. 聊天区域 (ChatArea)
*   **中间面板**：AI 对话的核心交互区。
*   **功能**：消息列表展示、输入框、发送按钮。
*   **响应式**：自适应宽度。

### 3. 代码预览区 (Preview Area)
*   **右侧面板**：实时代码预览。
*   **集成**：Sandpack (CodeSandbox) 运行时环境。
*   **功能**：实时编译和展示 React 代码。

## UI 优化
*   **持久化**：面板宽度调整后自动保存到 localStorage，刷新不丢失布局。
*   **响应式计算**：使用 ResizeObserver 或 CSS 媒体查询适配不同屏幕尺寸，聊天框宽度 dynamically calculated based on percentage for better UX.
*   **主题**：支持 Shadcn UI 的主题系统（默认暗色/亮色切换基础）。
