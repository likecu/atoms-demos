# 3.1 体验优化 (Optimization)

**目标**: 提升应用的交互细节和稳定性。

## 任务清单

- [ ] **1. Loading 状态反馈**
  - [ ] 在 AI 生成代码时，Chat 区域显示具体的阶段 (Streaming status)。
  - [ ] Sandpack 加载预览时显示 Skeleton 或 Loading Spinner。

- [ ] **2. 错误处理**
  - [ ] AI 请求超时/失败的 Toast 提示。
  - [ ] Sandpack 编译错误的界面友好显示 (Error Overlay 样式优化)。

- [ ] **3. UI 细节**
  - [ ] 按钮 Hover 态、点击态微调。
  - [ ] 滚动条样式美化 (Tailwind scrollbar plugin)。
  - [ ] 字体统一 (Inter / JetBrains Mono)。

## 验证步骤
1. 断网测试，确认错误提示。
2. 观察长文本生成时的滚动流畅度。
