# 1.2 布局重构 (Layout Refactoring)

**目标**: 调整现有双栏布局为三栏布局 (Sidebar | Chat | Preview)，并优化响应式体验。

## 任务清单

- [ ] **1. 组件拆分**
  - [ ] 确认 `ChatLayout` (`src/components/layout/chat-layout.tsx`) 当前结构。
  - [ ] 提取侧边栏组件: `src/components/layout/sidebar.tsx`。

- [ ] **2. 三栏布局实现**
  - [ ] 修改 `ChatLayout` 使用 `ResizablePanelGroup` 包含三个面板：
    - Panel 1 (Sidebar): defaultSize=20, minSize=15, collapsible=true.
    - Panel 2 (Chat): defaultSize=30, minSize=25.
    - Panel 3 (Preview): defaultSize=50, minSize=30.
  - [ ] 确保移动端适配 (在小屏幕下隐藏 Sidebar 和 Preview，只显示 Chat，或使用 Drawer)。

- [ ] **3. Sidebar 内容占位**
  - [ ] 在 `Sidebar` 组件中添加 "New Chat" 按钮。
  - [ ] 添加 "History" 列表的骨架屏或静态数据占位。
  - [ ] 添加 "User Profile" 底部区域 (显示头像/退出按钮)。

## 文件清单
- `src/components/layout/chat-layout.tsx`
- `src/components/layout/sidebar.tsx` (新建)

## 验证步骤
1. 桌面端打开，应看到三栏结构。
2. 拖拽分割线，三个区域应能正常缩放。
3. 点击 Sidebar 的收起按钮，应能折叠侧边栏。
