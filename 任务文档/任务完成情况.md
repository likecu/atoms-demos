## 🎯 当前功能亮点

### 已完成的核心功能
1. **用户认证系统** - 极简登录/注册，支持用户会话管理
2. **项目管理** - 创建、删除、重命名项目，Dashboard展示项目列表
3. **AI代码生成** - 集成OpenRouter API，支持多步工具调用
4. **实时预览** - Sandpack集成，即时预览生成的代码
5. **沙盒环境** - Docker容器隔离，支持命令执行
6. **Web终端** - 支持用户和AI执行命令
7. **文件浏览** - 查看项目文件结构和内容
8. **发布分享** - 一键生成公开分享链接
9. **AI状态监控** - 实时展示AI思考和工具调用流程
10. **无限画布** - 支持拖拽和缩放的预览画布

### 技术栈
- **前端**: Next.js 14, React 19, TypeScript, Tailwind CSS, Shadcn UI
- **后端**: Next.js Server Actions, Supabase (PostgreSQL)
- **AI**: OpenRouter API, Vercel AI SDK
- **部署**: Docker, 远程服务器
- **代码预览**: @codesandbox/sandpack-react
- **状态管理**: React Hooks, Context API
- **UI组件**: Radix UI, Framer Motion, Lucide Icons
